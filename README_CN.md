# ChatWhiz
随时切换OpenAI、Deepseek等AI模型，本地存储API Key与对话内容，提供无缝且安全的对话体验。

## 特性
- 支持OpenAI、Deepseek、ChatGLM等AI模型（将陆续增加）
- 本地存储API Key与对话内容，确保数据安全
- 自定义颜色（仅限移动端）
